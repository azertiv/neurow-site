Fonctions

Le couteau suisse de l'IA dans Excel.

AI.Native n'est pas juste un "chat". C'est une suite complète de 14 fonctions spécialisées pour traiter du texte, structurer des données, traduire et automatiser vos tableaux.
Toutes pilotables directement depuis vos cellules.

Installer l'extension Voir le Wiki

Génération & Analyse

Créez du contenu, résumez des documents et analysez des informations textuelles.

Généraliste

AI.ASK

Posez n'importe quelle question à l'IA, avec ou sans contexte.

=AI.ASK("Analyse ce feedback"; A2)
Synthèse

AI.SUMMARIZE

Résumez le contenu d'une cellule ou d'une plage entière en quelques mots.

=AI.SUMMARIZE(A1:A50)
Traduction

AI.TRANSLATE

Traduisez massivement vos contenus dans n'importe quelle langue cible.

=AI.TRANSLATE(A2:A100; "es")

Données & Structure

Transformez le chaos en tableaux exploitables. Structurez, classez, extrayez.

Extraction

AI.EXTRACT

Extrayez des entités spécifiques (emails, prix, dates) d'un texte en vrac.

=AI.EXTRACT(A2; "emails")
Classification

AI.CLASSIFY

Catégorisez automatiquement des lignes selon vos propres labels.

=AI.CLASSIFY(B2; "Lead|Client|Spam")
Tableaux

AI.TABLE

Générez un tableau structuré complet à partir d'une simple description.

=AI.TABLE("Top 10 pays PIB")

Web & Qualité

Enrichissez vos données avec le web et assurez leur cohérence.

Recherche

AI.WEB

Interrogez le web en direct. Récupérez des infos à jour avec leurs sources.

=AI.WEB("CEO de Apple actuel";;1)
Nettoyage

AI.CLEAN

Normalisez les formats, supprimez les caractères parasites et corrigez la casse.

=AI.CLEAN(C2:C100)
Cohérence

AI.CONSISTENT

Harmonisez des listes de valeurs proches (ex: "USA", "U.S.A", "United States").

=AI.CONSISTENT(D2:D50)

Outils Excel

Des utilitaires pour vous faire gagner du temps sur Excel même.

Formules

AI.FORMULA

Décrivez ce que vous voulez calculer, obtenez la formule Excel native.

=AI.FORMULA("Moyenne colonne A si B > 10")
Remplissage

AI.FILL

Complétez intelligemment une colonne en suivant quelques exemples donnés.

=AI.FILL(A1:B3; A4:A20; "Format...")
Diagnostic

AI.TEST / STATUS

Vérifiez vos clés API, le modèle actif et le statut des quotas.

=AI.KEYSTATUS()

Vie privée : pas de serveur intermédiaire.

AI.Native est conçu pour minimiser l’exposition des données. Vous fournissez votre clé API, et les appels partent directement d’Excel vers l’API du fournisseur choisi (OpenAI ou Gemini).

Flux direct Excel → API

Aucune donnée ne transite par un serveur AI.Native. Les informations envoyées sont uniquement celles que vous mettez dans la formule (texte, cellules, plages). Les clés API, le cache et les logs restent stockés localement sur votre appareil.

Clé API : vous l’apportez, vous la gardez.
Nettoyage : vous pouvez vider cache et logs depuis le panneau.
Transparence : les coûts API restent chez le fournisseur.
Voir les tarifs Obtenir une clé API
Note. Les données transmises sont ensuite traitées par le fournisseur (OpenAI ou Google) conformément à ses propres conditions et politiques.
