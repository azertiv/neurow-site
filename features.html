Features

Tout ce qu’il faut pour mettre l’IA au cœur d’Excel.

AI.Native transforme Excel en environnement IA “formula-first” : vous écrivez AI.* comme une fonction classique, vous choisissez le fournisseur (OpenAI / Gemini) et le modèle dans le panneau, et vous gardez la maîtrise des coûts via votre propre clé API.

En pratique

Vous gardez le contrôle

L’extension est volontairement “config-driven” : les choix importants se font dans le panneau (fournisseur, modèle, limites), pas dans chaque formule.

  • Un classeur reste lisible, même partagé.
  • Les réglages peuvent être ajustés sans réécrire les formules.
  • Vos données partent directement d’Excel vers l’API du fournisseur.

Une bibliothèque de fonctions IA pour vos workflows.

Les fonctions AI.* couvrent les cas d’usage Excel les plus fréquents : rédaction, classification, extraction, génération de formules, structuration en tableau… et recherche web quand vous en avez besoin.

Texte & analyse

Rédiger, expliquer, résumer

Produisez du texte exploitable à partir de cellules, plages ou consignes.

  • AI.ASK — question/réponse avec contexte
  • AI.SUMMARIZE — synthèse d’un texte ou d’un tableau
  • AI.CLEAN — nettoyage et normalisation de texte
Données

Extraire & classer

Transformez des données semi-structurées en colonnes utilisables.

  • AI.EXTRACT — extraction de champs
  • AI.CLASSIFY — catégories, tags, labels
  • AI.TABLE — tableau “prêt Excel” en sortie
Automatisation Excel

Formules & remplissage

Accélérez la construction de modèles et la mise en forme des sorties.

  • AI.FORMULA — génère une formule à partir d’une intention
  • AI.FILL — complétion/réécriture sur une plage
  • AI.CONSISTENT — cohérence et normalisation

Recherche web intégrée, avec des sources.

Besoin d’une information externe dans un tableau Excel ? AI.WEB permet d’interroger le web et de récupérer une réponse exploitable, avec affichage optionnel des sources.

AI.WEB

Réponse + liens

Conçu pour la veille, les benchmarks, la vérification rapide et la génération de notes avec sources.

  • Réponse adaptée à une cellule ou une matrice “spill”.
  • Sources listées sous forme de liens (quand activé).
Structuration

Pensé pour Excel

L’objectif n’est pas “un chat dans Excel”, mais des sorties stables et réutilisables dans un modèle.

  • Moins d’options dans les formules : configuration centralisée.
  • Réduction des prompts répétitifs grâce au panneau.

Vie privée : pas de serveur intermédiaire.

AI.Native est conçu pour minimiser l’exposition des données. Vous fournissez votre clé API, et les appels partent directement d’Excel vers l’API du fournisseur choisi (OpenAI ou Gemini).

Flux direct Excel → API

Aucune donnée ne transite par un serveur AI.Native. Les informations envoyées sont uniquement celles que vous mettez dans la formule (texte, cellules, plages). Les clés API, le cache et les logs restent stockés localement sur votre appareil.

Clé API : vous l’apportez, vous la gardez.
Nettoyage : vous pouvez vider cache et logs depuis le panneau.
Transparence : les coûts API restent chez le fournisseur.
Voir les tarifs Obtenir une clé API
Note. Les données transmises sont ensuite traitées par le fournisseur (OpenAI ou Google) conformément à ses propres conditions et politiques.
