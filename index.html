Le Couteau Suisse

=AI.ASK(Prompt; [Contexte])

L'équivalent d'un chat dans une cellule. Parfait pour rédiger, résumer ou analyser un contenu spécifique.

=AI.ASK("Résume ce texte en 3 points"; A2)
Exclusivité

=AI.WEB(Recherche; [Focus])

Récupérez des données fraîches du web (Google Search). Idéal pour l'enrichissement de données et la veille.

=AI.WEB("CA 2024 Tesla";;1)
Organisation

=AI.CLASSIFY(Texte; Catégories)

Classifie automatiquement vos lignes. Détectez le sentiment, triez des tickets ou segmentez des leads.

=AI.CLASSIFY(B2; "Urgent|Normal|À ignorer")
Extraction

=AI.EXTRACT(Texte; Champs)

Transforme du texte non structuré (emails, descriptions) en données exploitables.

=AI.EXTRACT(A2; "Email, Date, Montant")

FAQ

Les questions qui reviennent avant installation.

Mes données sont-elles confidentielles ?

Absolument. Neurow est conçu sur une architecture "No Backend". Vos données transitent directement de votre tableau Excel vers l'API du fournisseur (OpenAI ou Google) sans jamais passer par nos serveurs intermédiaires.

  • Vos clés API sont stockées localement sur votre ordinateur (cryptées).
  • L'historique et le cache sont conservés uniquement sur votre machine.
  • C'est vous qui détenez la relation avec le fournisseur d'IA.
Quels modèles d'IA puis-je utiliser ?

Vous avez le contrôle total et accès aux modèles les plus récents. Le choix se fait en un clic dans le panneau latéral, sans toucher à vos formules.

  • Google Gemini : Gemini 3.0 Flash (recommandé pour la vitesse), Gemini 3.0 Pro, Gemini 2.5 Flash-Lite.
  • OpenAI : GPT-5 Mini (recommandé), GPT-5.2, GPT-5 Nano.
  • Vous pouvez également saisir manuellement tout nouveau modèle compatible dès sa sortie.
Ai-je besoin d'un abonnement ChatGPT ou Gemini ?

Non, les abonnements "Grand Public" (type ChatGPT Plus) ne sont pas utilisés. Vous avez besoin d'une Clé API.

  • Chez Google : Les clés API (Google AI Studio) offrent souvent un quota gratuit généreux, idéal pour démarrer.
  • Chez OpenAI : Il s'agit d'un système de crédits prépayés (Pay-as-you-go). Vous ne payez que ce que vous consommez réellement (souvent quelques centimes pour des milliers de cellules).
Quelle différence avec Microsoft Copilot ?

Copilot est un assistant conversationnel généraliste. Neurow est une boîte à outils pour bâtisseurs.

  • Précision : Utilisez des fonctions dédiées (AI.EXTRACT, AI.CLASSIFY) directement dans vos flux de données.
  • Cache Intelligent : Neurow garde en mémoire les résultats pour ne pas repayer si vous relancez un calcul.
  • Flexibilité : Choisissez le modèle exact, la "température" et l'effort de raisonnement adaptés à votre métier.
Quelles sont les fonctions incluses ?

Neurow ajoute une suite de fonctions natives à Excel :

  • =AI.ASK(...) : Instructions libres.
  • =AI.WEB(...) : Recherche web en temps réel avec citations de sources.
  • =AI.TRANSLATE(...) : Traduction rapide de cellules ou plages.
  • =AI.CLASSIFY(...) : Catégorisation automatique (Lead, Ticket, Sentiment...).
  • =AI.EXTRACT(...) : Extraction structurée (Emails, Dates, Montants).
Est-ce compatible avec mon ordinateur ?

Oui. Neurow est un "Office Add-in" moderne. Il fonctionne partout où Excel moderne fonctionne :

  • Windows / Mac : Excel Desktop.
  • Web : Excel Online (dans votre navigateur).

L'installation se fait en un clic via le Microsoft Store, sans droits d'administrateur complexes.
