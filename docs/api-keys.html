Obtenir une clé API (OpenAI & Gemini)

AI.Native fonctionne avec votre clé API. Cette page explique comment créer une clé en quelques minutes, même si vous débutez.

Sécurité. Une clé API est un secret. Ne la collez jamais dans une cellule Excel, ne la partagez pas, et évitez de la stocker dans des documents synchronisés/publics. Utilisez uniquement le panneau AI.Native.

Clé API OpenAI

AI.Native utilise l’API OpenAI (endpoint /v1/responses). Vous avez besoin d’une clé commençant généralement par sk-....

Étapes (débutant)

  1. Ouvrez la plateforme OpenAI :
https://platform.openai.com/
  1. Connectez-vous (ou créez un compte).
  2. Dans le dashboard, ouvrez la section API keys.
  3. Cliquez sur Create new secret key (le libellé peut varier) et copiez la clé.
  4. Si la plateforme le demande, configurez la facturation et/ou des limites d’usage (recommandé).
  5. Dans Excel → panneau AI.Native → onglet OpenAI :
    • Choisissez le modèle.
    • Collez la clé dans “Clé API OpenAI”.
    • Cliquez Enregistrer puis Tester.

Référence officielle (quickstart) :

https://platform.openai.com/docs/quickstart

Bonnes pratiques

  • Commencez avec un modèle “mini” pour maîtriser la dépense, puis montez en gamme si nécessaire.
  • Surveillez vos quotas (un recalcul Excel peut relancer des appels si vous modifiez des cellules).
  • Rotation : si vous suspectez une fuite, révoquez la clé et recréez-en une.

Clé API Gemini (Google AI Studio)

AI.Native utilise l’API Gemini via la Generative Language API. Une clé Gemini a souvent un format du type AIza....

Étapes (débutant)

  1. Ouvrez Google AI Studio :
https://aistudio.google.com/
  1. Connectez-vous avec votre compte Google.
  2. Accédez à la page API Keys puis cliquez Create API key.
  3. Choisissez un projet Google Cloud existant ou créez-en un si besoin.
  4. Copiez la clé.
  5. Dans Excel → panneau AI.Native → onglet Gemini :
    • Choisissez le modèle.
    • Collez la clé dans “Clé API Gemini”.
    • Cliquez Enregistrer puis Tester.

Référence officielle Google (clés Gemini API) :

https://ai.google.dev/gemini-api/docs/api-key

Vérifier que la clé est bien prise en compte

Dans Excel :

=AI.KEYSTATUS()

Puis :

=AI.TEST()
En cas d’erreur. Allez sur Dépannage et vérifiez : modèle sélectionné, quotas/crédit, effort de raisonnement, et réseau.
