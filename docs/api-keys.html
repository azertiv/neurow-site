Obtenir une clé API

AI.Native nécessite votre propre clé pour dialoguer avec les modèles d'IA. C’est ce qui vous garantit la confidentialité (vos données partent directement chez le fournisseur, sans passer par nous) et le contrôle des coûts.

Sécurité importante : Une clé API est un mot de passe. Ne la partagez jamais avec personne. Si vous pensez qu'une clé a été compromise, supprimez-la immédiatement sur le site du fournisseur.

OpenAI

Pour utiliser GPT-4o, GPT-5 Mini, etc. Idéal pour des tâches complexes et la précision maximale.

Google Gemini

Pour utiliser Gemini 1.5 Flash, Gemini 3 Pro, etc. Souvent plus rapide et avec une offre gratuite généreuse.

Configurer Gemini ↓

1. Obtenir une clé OpenAI

Suivez ces étapes pour générer une clé commençant par sk-....

1. Accédez à la plateforme OpenAI

Rendez-vous sur platform.openai.com/api-keys et connectez-vous avec votre compte ChatGPT/OpenAI.

2. Créez une nouvelle clé secrète

Dans la section API keys, localisez le bouton de création.

platform.openai.com — API keys
API keys
sk-...Ms92 Sept 22, 2024
+ Create new secret key
3. Copiez la clé immédiatement

Une fenêtre s’affiche avec votre clé (sk-...). Copiez-la maintenant car vous ne pourrez plus la revoir ensuite. Si votre compte est nouveau, vous devrez peut-être ajouter un moyen de paiement dans Settings > Billing pour activer l'accès (les crédits gratuits expirent souvent vite).

4. Collez la clé dans Excel

Ouvrez Excel et le panneau AI.Native.

2. Obtenir une clé Google Gemini

L'API Gemini est souvent gratuite pour un usage modéré (Free Tier). La clé commence généralement par AIza....

1. Accédez à Google AI Studio

Allez sur aistudio.google.com/app/apikey et connectez-vous avec votre compte Google.

2. Cliquez sur "Get API key"

Dans le menu de gauche ou sur l'écran principal, cherchez le bouton pour créer une clé.

Google AI Studio
Home
Get API key
API keys
API Key 1
AIzaSyD...
3. Création dans un projet

Google vous demandera de lier la clé à un projet Google Cloud. Vous pouvez choisir "Create API key in new project" pour faire simple.

4. Configuration dans Excel

Copiez la clé AIza... et retournez dans le panneau AI.Native.

Vérifier que tout fonctionne

Une fois la clé enregistrée, testez immédiatement la connexion dans une cellule Excel vide :

STATUT
fx =AI.KEYSTATUS()
1
[OK] Key configured (sk-...)

Vérifie que le format de la clé est correct et qu'elle est bien sauvegardée localement.

TEST RÉSEAU
fx =AI.TEST()
1
Connection Successful! (gpt-4o)

Effectue un véritable appel API (très court) pour vérifier que la clé est valide et le crédit suffisant.
