Installation & première formule

Objectif : installer Neurow, configurer une clé API, vérifier la connexion, puis exécuter une première fonction.

1) Installer l’extension

  1. Ouvrez Excel.
  2. Allez dans Insertion → Mes compléments (ou Obtenir des compléments).
  3. Recherchez Neurow dans le Microsoft Store et installez.
Note. Neurow s’appuie sur un Shared Runtime : le panneau et les fonctions personnalisées tournent dans le même runtime et partagent configuration et cache.

2) Ouvrir le panneau et configurer

  1. Depuis le ruban Excel, cliquez sur Neurow.
  2. Choisissez votre fournisseur par défaut (Gemini ou OpenAI).
  3. Sélectionnez un modèle et collez votre clé API.
  4. Ajustez Max output tokens (conseil : 256–2048 pour démarrer).
  5. Cliquez Enregistrer puis Tester.

Si vous n’avez pas encore de clé API, suivez : Obtenir une clé API (OpenAI & Gemini).

3) Valider que tout est OK

Dans une cellule :

=AI.KEYSTATUS()

Vous devez voir le fournisseur par défaut, les modèles et l’état des clés.

=AI.TEST()

Cette formule effectue un test minimal. Si elle renvoie OK, vous êtes prêt.

4) Votre première formule “utile”

Exemples (Excel FR) :

=AI.ASK("Explique la variance en 3 puces"; A1:B5)
=AI.TRANSLATE(A2:A10; "en")
=AI.CLASSIFY(B2:B20; "Urgent|Normal|À traiter")

Bonnes pratiques (très importantes dans Excel)

  • Batch & spill : privilégiez les fonctions qui traitent une plage en une seule requête (traduction, classification, nettoyage…).
  • Cache : laissez le cache actif si vos prompts se répètent (séries de calcul, recalculs Excel).
  • Contexte : passez une plage de contexte quand cela apporte une vraie information (ex : noms de colonnes, exemples).
  • Max tokens : augmentez uniquement si nécessaire (réponses longues = coûts et latence).
Attention à la sensibilité des données. Même sans serveur intermédiaire, vos textes et cellules envoyés au modèle sont traités par le fournisseur (OpenAI/Gemini). Ne transmettez pas d’informations si votre politique interne l’interdit.
