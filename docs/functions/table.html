AI.TABLE

Signature. =AI.TABLE(description; [contextRange])

Génère un tableau (en-têtes + lignes) à partir d’une description, avec sortie structurée.

Sortie

Matrice (spill) avec une première ligne d’en-têtes.

Exemples (Excel FR)

=AI.TABLE("Top 5 villes de France avec population")
=AI.TABLE("Tableau des KPI : nom, définition, formule Excel")
=AI.TABLE("Plan de projet (phase, livrable, date)"; A1:B10)

Conseils

  • Parfait pour démarrer une table structurée à partir d’un brief.
  • Ajoutez un contexte (ex. colonnes existantes) pour aligner le tableau à votre modèle.
  • Si le tableau déborde, réduisez la description ou le max tokens.

Notes

  • Si l’IA ne renvoie pas d’en-têtes, Neurow génère des en-têtes par défaut (Col1, Col2, …).
