AI.CLASSIFY

Signature. =AI.CLASSIFY(textOrRange; labels)

Classe chaque texte dans un label strict parmi une liste donnée.

Sortie

Matrice (spill) alignée sur l’entrée.

Exemples (Excel FR)

=AI.CLASSIFY(B2; "Urgent|Normal|À traiter")
=AI.CLASSIFY(B2:B100; "Positif|Neutre|Négatif")
=AI.CLASSIFY(C2:C50; A1:A5)

Conseils

  • Donnez des labels mutuellement exclusifs et lisibles.
  • Les labels peuvent être passés en chaîne (séparateurs : |, ;, ,) ou via une plage.
  • Prévoyez un label “Autre” / “UNKNOWN” si vous avez des cas hors distribution.

Notes

  • La fonction tente de renvoyer exactement l’un des labels fournis.
