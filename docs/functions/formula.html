AI.FORMULA

Signature. =AI.FORMULA(description; [contextRange])

Génère une formule Excel valide à partir d’une description. Utilise automatiquement les noms de fonctions FR/EN selon la locale Excel.

Sortie

Texte (scalaire), commençant par =.

Exemples (Excel FR)

=AI.FORMULA("Sommer les ventes par région"; A1:C200)
=AI.FORMULA("Extraire le mois d une date"; A2)
=AI.FORMULA("Top 3 valeurs uniques de A2:A200")

Conseils

  • Décrivez les colonnes dans le contexte (en-têtes) pour de meilleures formules.
  • Si votre Excel est en français, la formule utilisera des noms FR et ; comme séparateur.
  • Après génération, relisez et testez : une formule peut être valide mais ne pas refléter exactement votre intention.

Notes

  • La fonction renvoie uniquement la formule, sans texte additionnel.
