AI.SUMMARIZE

Signature. =AI.SUMMARIZE(textOrRange; [mode])

Résume un texte ou une plage. En mode 0 (défaut), résume chaque cellule indépendamment. En mode 1, produit un résumé global de la plage.

Sortie

Matrice (spill). En mode 1, renvoie une cellule (1×1).

Exemples (Excel FR)

=AI.SUMMARIZE(A2)
=AI.SUMMARIZE(A2:A20)
=AI.SUMMARIZE(A2:A20; 1)

Conseils

  • Mode 0 pour des résumés ligne à ligne ; mode 1 pour une synthèse globale (notes de réunion, retours clients, etc.).
  • Demandez des puces courtes dans le texte d’entrée si nécessaire (ou passez par AI.ASK).
  • Réduisez le volume (ou filtrer) avant d’agréger des milliers de lignes.
