AI.FILL

Signature. =AI.FILL(exampleRange; targetRange; instruction)

Remplit une colonne cible à partir d’exemples (apprentissage par démonstration).

Sortie

Colonne (spill) alignée sur targetRange (une valeur par ligne).

Exemples (Excel FR)

=AI.FILL(A2:B6; D2:D20; "Normaliser en nom complet")
=AI.FILL(A2:B10; C2:C100; "Extraire la catégorie produit")

Conseils

  • Le format attendu est généralement 2 colonnes d’exemples : [entrée | sortie].
  • Plus vos exemples couvrent les cas variés, plus le remplissage sera stable.
  • Ajoutez une instruction de format (ex. “en MAJUSCULE”, “format ISO date”).
