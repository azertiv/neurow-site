AI.EXTRACT

Signature. =AI.EXTRACT(textOrRange; instruction)

Extrait des informations d’un texte selon une instruction concise (ex : “emails”, “noms”, “actions”, “dates”).

Sortie

Si l’entrée est une cellule : liste verticale (spill) possible. Si l’entrée est une plage : matrice alignée.

Exemples (Excel FR)

=AI.EXTRACT(A2; "emails")
=AI.EXTRACT(A2:A50; "numéros de commande")
=AI.EXTRACT(B2:B10; "actions à faire")

Conseils

  • Pour une cellule, AI.EXTRACT peut renvoyer plusieurs éléments (spill vertical).
  • Pour une plage, AI.EXTRACT renvoie un élément “meilleur effort” par cellule.
  • Soyez explicite dans l’instruction (format souhaité, langue, précision).

Notes

  • La sortie est conçue pour être structurée et éviter les hallucinations : privilégiez des instructions de type “extraire X”.
