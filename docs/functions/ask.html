AI.ASK

Signature. =AI.ASK(prompt; [contextRange])

Pose une question ou donne une instruction à l’IA. Le fournisseur, le modèle, et le max tokens sont définis dans le panneau.

Sortie

Texte (scalaire).

Exemples (Excel FR)

=AI.ASK("Explique la variance en 3 puces")
=AI.ASK("Résume les points clés"; A1:D20)
=AI.ASK("Génère 5 idées de titres"; A2:A20)

Conseils

  • Utilisez contextRange pour fournir des exemples, des colonnes, ou un extrait (TSV) plutôt que tout le fichier.
  • Si la réponse est trop longue, baissez Max output tokens ou demandez un format concis.
  • Pour des listes, demandez explicitement des puces courtes (Excel).

Notes

  • Neurow supprime certains marqueurs Markdown simples pour faciliter l’affichage en cellule.
