AI.WEB

Signature. =AI.WEB(query; [focus]; [showSource])

Recherche factuelle. Active les capacités de recherche web du fournisseur (si disponibles) et peut afficher des sources.

Sortie

Texte (scalaire). Option showSource=1 pour inclure une liste de sources.

Exemples (Excel FR)

=AI.WEB("Population France 2024";;1)
=AI.WEB("Chiffre d’affaires Apple 2024";;1)
=AI.WEB("Date de naissance Marie Curie";;1)

Conseils

  • Soyez précis (pays, année, périmètre) pour une réponse plus fiable.
  • Utilisez focus pour cadrer l’entité ou le contexte (ex. nom de l’entreprise + pays).
  • Si vous voyez une justification au lieu d’une valeur, la donnée n’a pas pu être confirmée de manière fiable.

Notes

  • Le résultat est conçu pour être court et sourcé. Les résultats web évoluent : Neurow applique un TTL plus court sur le cache web.
