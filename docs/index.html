Wiki AI.Native

Bienvenue. Ce wiki décrit l’installation, la configuration (fournisseur, modèle, tokens, effort de raisonnement) et la référence complète des fonctions AI.*.

Principe clé. Le fournisseur (Gemini / OpenAI) se choisit dans le panneau AI.Native, pas dans les formules. L’objectif : des fichiers lisibles et des formules stables.

En 2 minutes : votre première formule

  1. Installez l’add-in depuis le Microsoft Store.
  2. Ouvrez le panneau AI.Native (ruban Excel).
  3. Choisissez le fournisseur, entrez votre clé API, cliquez Tester.
  4. Dans une cellule : =AI.ASK("Explique la variance"; A1:B5)

Installation & première formule Obtenir une clé API

Ce que fait AI.Native (selon votre code)

  • Appels directs aux API : Gemini via l’endpoint generativelanguage.googleapis.com, OpenAI via /v1/responses.
  • Cache local pour éviter les appels répétés (donc moins de tokens) ; expiration automatique (TTL).
  • Déduplication des appels identiques “en vol” et limite de concurrence (anti-burst).
  • Sorties structurées pour rendre les matrices “spill” robustes et prévisibles dans Excel.
  • AI.WEB active les outils de recherche web natifs (quand supportés) et peut afficher des sources.
