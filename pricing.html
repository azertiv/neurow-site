Question fréquente

Pourquoi un abonnement si j’apporte ma clé API ?

Parce que la valeur est dans l’expérience Excel : fonctions pensées pour les feuilles (spill, batch, formules), UI de configuration, cache, logs, déduplication, et une intégration propre “sans options dans la formule”.
