Politique de confidentialité

Dernière mise à jour : 30 Décembre 2025

Architecture "No Backend"

Neurow est une extension "Client-Side Only". Nous ne possédons aucun serveur pour traiter vos données. Vos fichiers Excel, vos clés API et vos prompts ne transitent jamais par nos systèmes. L'extension fait le lien direct entre votre Excel et les fournisseurs d'IA (Google/OpenAI).

1. Introduction

La présente politique de confidentialité décrit comment l'extension Neurow (éditée par Neurow) traite, protège et utilise vos données. Notre philosophie est simple : vos données vous appartiennent, et nous avons conçu l'architecture de Neurow pour qu'il soit techniquement impossible pour nous d'y accéder.

2. Collecte et Transmission des Données

2.1 Données que vous fournissez volontairement

Pour utiliser Neurow, vous devez fournir vos propres clés API (Google Gemini ou OpenAI). Ces clés sont utilisées pour authentifier les requêtes que vous lancez depuis Excel.

2.2 Transmission aux Fournisseurs Tiers (AI Providers)

Lorsque vous exécutez une fonction (par exemple =AI.ASK(...)), Neurow transmet uniquement :

  • Le prompt (texte) contenu dans la formule.
  • Le contexte éventuel (plage de cellules sélectionnée).
  • Les paramètres techniques (température, modèle choisi).

Ces données sont envoyées directement aux API de Google (Gemini) ou OpenAI via une connexion chiffrée (HTTPS). Aucune de ces données ne passe par un serveur contrôlé par Neurow.

3. Stockage Local (Sur votre appareil)

Neurow privilégie le stockage local pour garantir votre sécurité et améliorer les performances.

3.1 Clés API

Vos clés API sont stockées exclusivement sur votre appareil, via les mécanismes sécurisés du navigateur ou d'Office (OfficeRuntime.storage ou localStorage). Elles ne sont jamais envoyées ailleurs qu'aux serveurs d'authentification des fournisseurs respectifs (Google ou OpenAI).

3.2 Cache et Logs

Pour éviter de refacturer des appels identiques et pour vous permettre de suivre votre consommation :

  • Cache : Les réponses des IA sont mises en cache localement sur votre machine (avec une durée de vie limitée) pour accélérer l'affichage.
  • Logs d'utilisation : Un historique technique (nombre de tokens, latence, coût estimé, erreurs) est conservé localement pour alimenter votre tableau de bord.

Vous avez le contrôle total sur ces données et pouvez les effacer à tout moment via le bouton "Vider le cache" ou "Effacer" dans le panneau de configuration.

4. Note importante sur les services tiers

L'utilisation de Neurow implique l'envoi de données aux fournisseurs d'IA que vous choisissez (Google Gemini ou OpenAI). Ces échanges sont régis par leurs conditions respectives.

Google Privacy Policy ↗ OpenAI Privacy Policy ↗

Nous vous conseillons de vérifier que vos clés API sont configurées pour ne pas autoriser l'entraînement des modèles sur vos données (option généralement active par défaut pour les API payantes).

5. Absence de système de suivi (Analytics)

Neurow n'intègre aucun traceur publicitaire ni aucun outil d'analyse comportementale (type Google Analytics) qui enverrait des données sur votre usage à nous ou à des tiers.

6. Modification et Contact

Nous pouvons mettre à jour cette politique pour refléter des changements techniques (nouveaux fournisseurs supportés). Puisque nous n'avons pas votre email (nous ne gérons pas de comptes utilisateurs), nous vous invitons à consulter cette page régulièrement.
