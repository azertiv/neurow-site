Politique de confidentialité (modèle)

Ce document est un modèle à adapter avant publication sur le Microsoft Store.

1) Données traitées par l’extension

AI.Native traite les données nécessaires à l’exécution des fonctions IA dans Excel : le texte de vos prompts, et éventuellement des valeurs de cellules (plages) que vous fournissez en contexte.

2) Absence de serveur AI.Native

AI.Native ne met pas en place de serveur intermédiaire. Les requêtes sont envoyées directement depuis votre environnement Office vers le fournisseur choisi (OpenAI ou Google Gemini).

3) Stockage local

  • Les clés API sont stockées localement (OfficeRuntime.storage ou localStorage).
  • AI.Native peut stocker localement un cache de réponses et des logs techniques (diagnostic).

4) Partage avec des tiers

Les données envoyées via les appels API sont partagées avec le fournisseur choisi (OpenAI ou Google) conformément à leurs conditions et politiques. AI.Native ne revend pas ces données et ne les traite pas sur un serveur propre.

5) Gestion et suppression

Vous pouvez supprimer les caches et logs depuis le panneau AI.Native. Vous pouvez effacer vos clés API depuis le panneau.
